About Movie API

This API will provide information on movies including but not limited to their descriptions, genres and directors. It will also allow users to register/deregister and add/remove movies from a favorites list

Url endpoints and requrest methods methods

  • Return list of movies
    • url: /movies
    • Request type: GET
    • provides JSON data of all movies
  • Return info on a movie by name
    • url: /movies/:Title
    • Request type: GET
    • provides JSON data for a specific movie by name
    • Example:
      "Genre": {
      "Name": "Science Fiction",
      "Description": "Speculative fiction that contains imagined elements that do not exist in the real world."
      },
      "Director": {
      "Name": "James Cameron",
      "Bio": "James Francis Cameron is a Canadian filmmaker. Best known for making science fiction and epic films. Popular films include Avatar and Titanic.",
      "Birth": "1954"
      },
      "_id": "61eda33c2db24d3bf8793079",
      "Title": "Avatar",
      "Description": "A paraplegic Marine dispatched to the moon Pandora on a unique mission becomes torn between following his orders and protecting the world he feels is his home. When his brother is killed in a robbery, paraplegic Marine Jake Sully decides to take his place in a mission on the distant world of Pandora.",
      "ImagePath": "avatar.png",
      "Featured": true
  • Return info on genre by type
    • url: /genres/:Genre
    • Request type: GET
    • provides JSON data about a genre by type
    • Example:
      "Name": "Horror",
      "Description": "A genre in which the movie creates feelings of fear keeping the audience on the edge of their seats."
  • Return info on a director by name
    • url: /directors/:directorName
    • Request type: GET
    • provides JSON data on director information
    • Example:
      "Name": "James Cameron",
      "Bio": "James Francis Cameron is a Canadian filmmaker. Best known for making science fiction and epic films. Popular films include Avatar and Titanic",
      "Birth": "1954"
  • Allow new user to register
    • url: /users
    • Request type: POST
    • Returns JSON data of the new users information
    • Example:
      "Username": "User11",
      "Password": "Password11!",
      "Email": "User11@gmail.com",
      "FavoriteMovies": [],
      "_id": "61f1b82087c59aa5fabb8ecb",
      "__v": 0
  • Allow user to update username
    • url: /users/:Username
    • Request type: PUT
    • Returns confirmation when successful
    • Example:
      "_id": "61f1b82087c59aa5fabb8ecb",
      "Username": "user011",
      "Password": "Password11!",
      "Email": "User11@gmail.com",
      "FavoriteMovies": [],
      "__v": 0
  • Allow user to add movie to favorites list
    • url: /users/:Username/favorites/:MovieID
    • Request type: POST
    • Returns JSON data on updated user data including the added movie
    • Example:
      "_id": "61f1b82087c59aa5fabb8ecb",
      "Username": "user011",
      "Password": "Password11!",
      "Email": "User11@gmail.com",
      "FavoriteMovies": [
      "61eda33c2db24d3bf8793079"
      ],
      "__v": 0
  • Allow user to remove movie from favorites list
    • url: /user/:Username/favorites/:MovieID
    • Request type: DELETE
    • Returns JSON data on updated user information, excluding the deleted movie
    • Example:
      "_id": "61f1b82087c59aa5fabb8ecb",
      "Username": "user011",
      "Password": "Password11!",
      "Email": "User11@gmail.com",
      "FavoriteMovies": [],
      "__v": 0
  • Allow user to deregister
    • url: /users/:Username
    • Request type: DELETE
    • Returns confirmation that the user was removed
    • Example:
      user011 has been deleted.
