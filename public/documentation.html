About Movie API

This API will provide information on movies including but not limited to their descriptions, genres and directors. It will also allow users to register/deregister and add/remove movies from a favorites list

Url endpoints and requrest methods methods

  • Return list of movies
    • url: /movies
    • Request type: GET
    • provides JSON data of all movies
  • Return info on a movie by name
    • url: /movies/:title
    • Request type: GET
    • provides JSON data for a specific movie by name
  • Return info on genre by type
    • url: /genres/:type
    • Request type: GET
    • provides JSON data about a genre by type
  • Return info on a director by name
    • url: /directors/:name
    • Request type: GET
    • provides JSON data on director information
  • Allow new user to register
    • url: /users
    • Request type: POST
    • Returns confirmation when successful
  • Allow user to update username
    • url: /users/:username
    • Request type: PUT
    • Returns confirmation when successful
  • Allow user to add movie to favorites list
    • url: /users/:username/favorites/:title
    • Request type: POST
    • Returns confirmation when successful
  • Allow user to remove movie from favorites list
    • url: /user/:username/favorites/:title
    • Request type: DELETE
    • Returns confirmation when successful
  • Allow user to deregister
    • url: /users
    • Request type: DELETE
    • Returns confirmation when successful
